Whydah

Whydah is a set of open source light weight user administration and single sign on components developed in Oslo, Norway and supported by Cantara.

It contains four logical components built using Java. Documentation is found at wiki.cantara.no/display/whydah

This component, a simple user- and application administration tool, is built to make it easy to browse and modify users.

This component is, in addition to Java, built with:

  • Angular.js
  • Bootstrap / Bootswatch Yeti
  • jQuery
  • Components from UI Bootstrap
